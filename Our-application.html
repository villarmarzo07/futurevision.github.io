GUI handling instructions

1. Loading and saving an image

On the top toolbar, in the file menu, select the option "Open" to load a file. The file menu also allows the user to save the result image using the option "Save".

2. Selecting the detection methods

On the top toolbar, there are another two menus that are used to select the detection methods for both the car and the license plates. 

3. Starting the detection

The user can begin the detection by pressing the "Start detection" button. Once the algorithm has finished, the cars and license plates bounding boxes are drawn onto the original image.
